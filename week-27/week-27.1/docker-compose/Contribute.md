## Manual Installation
 - Install nodejs 
 - Clone the repo
 - Install dependencies (npm i)
 - Start DB locally
    - docker run --name postgres-db -e POSTGRES_PASSWORD=postgres -p 5432:5432 -d postgres
    - Or go to neon.com and create a free db

 - Generate prisma client
    - npx prisma generate
 - Start the server
    - npm run dev
